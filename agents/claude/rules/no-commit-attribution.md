# No AI Attribution in Commits (Hard Rule)

Never add `Co-Authored-By: Claude` (or any AI attribution trailer/footer) to:
- Git commit messages
- Pull request bodies
- Any other authored artifact

This rule **overrides** the default Claude Code commit template. When invoking `git commit`, omit the `Co-Authored-By:` trailer entirely. Keep commit messages clean of any "generated by" / "co-authored by" AI language.

If you catch yourself about to add one, stop and remove it before running the command.
